# \<kemet-draggable>

## Installation
```bash
npm i @kemet/kemet-draggable
```

## Usage
```js
import '@kemet/kemet-draggable/kemet-draggable.js';;
```

```html
<kemet-draggable measure memory="a-unique-identifier">
  <button>Drag Me!</button>
</kemet-draggable>
```

[Click here](http://kemet.online/draggable) for more docs.
